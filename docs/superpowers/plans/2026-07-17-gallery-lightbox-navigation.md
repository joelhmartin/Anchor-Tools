# Gallery Lightbox Navigation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Clicking any item in an Anchor Gallery opens a lightbox that navigates through every visible item in that gallery — via arrows, keyboard, or swipe — stopping any playing video the moment you navigate away.

**Architecture:** On tile click, walk up to the `.anchor-video-gallery` container and collect the currently-visible tiles into an ordered sequence, then hand that array plus a start index to a rewritten `openLightbox(sequence, startIndex, opts)`. All rendering funnels through one `renderLightboxItem(i)` whose first action is `frame.innerHTML = ''`. Because playback is a raw autoplay iframe, destroying the frame *is* stopping the video — so the stop-on-navigate requirement is satisfied structurally, by having exactly one teardown path, rather than by remembering to stop playback at each call site.

**Tech Stack:** Vanilla ES5 JavaScript (no jQuery, no bundler), PHP 7.4+/WordPress, Playwright E2E on `@wordpress/env`.

**Spec:** `docs/superpowers/specs/2026-07-17-gallery-lightbox-navigation-design.md`

## Global Constraints

- **Vanilla ES5 only.** `var`, no arrow functions, no `const`/`let`, no template literals, no optional chaining. The module's frontend JS is two plain IIFEs with `'use strict'`. jQuery appears only as a duck-type check at `anchor-video-slider.js:1068`. Do not introduce jQuery or any dependency.
- **Never create, edit, or commit `*.min.css`, `*.min.js`, or `*.min.js.map`.** They are gitignored and regenerated by `bin/build-assets.mjs` in the release workflow. Edit only the source `.css` / `.js`.
- **No manual asset version bump.** Frontend gallery assets are versioned by `filemtime()` (`anchor-gallery.php:1431-1432`), so cache-busting is automatic. This overrides the generic "bump version string" convention in `CLAUDE.md` — it applies to modules that hardcode a version, not this one.
- **Scope is `popup_style = 'lightbox'` only.** Theater, side panel, and inline expand keep their current single-item behavior. Do not modify `openTheater`, `openSidePanel`, or `openInline`.
- **`logo_carousel` is out of scope.** It emits `.avg-marquee-item` with no data attributes and no click behavior.
- **Text domain** for any translatable string: `'anchor-schema'`.
- **Asset URLs** use `Anchor_Asset_Loader::url('anchor-gallery/assets/...')` — never `plugin_dir_url(__FILE__)`.
- **Wrap-around** at both ends: item N → item 1, item 1 → item N.
- Run E2E from the repo root. The environment must be up (`npm run wp-env start`) and seeded (`npm run env:seed`) before any spec runs.

## Key Facts (verified — do not re-derive)

- Playback is a raw iframe with `autoplay=1` in the query string (`js:8-16`). The YouTube IFrame Player API is **not** loaded, there is no `enablejsapi=1`, no `YT.Player`, no `postMessage`. **Do not add them.** Destroying the iframe is the stop mechanism, and it already works on close (`js:141-142`).
- **Filters hide via `.is-hidden`** (`js:1157`); **pagination hides via `.avg-hidden`** (`js:888`). Two different classes. Both resolve to `display:none`. Therefore visibility is tested with **`el.offsetParent !== null`**, which catches both without hardcoding either, and correctly does *not* treat carousel tiles as hidden (they keep layout boxes and merely scroll out of view, `css:570-582`).
- Tile markup (`anchor-gallery.php:2219-2255`):
  - HTML item: `<div class="avg-tile avg-tile-html" data-type="html">` → `.avg-tile-inner` → `.avg-html-content`. **Has no `data-caption`** (the caption attribute is computed after the `continue` at `:2228`).
  - Image item: `data-full-url`; alt text lives on `img.avg-thumb-img`.
  - Video item: `data-provider`, `data-video-id`, `data-url`, optional `data-duration`.
  - Linked item: `<a class="avg-tile avg-tile-linked">` — exists only when `popup_style === 'none'`, which never coexists with a lightbox. Excluded defensively.
- Gallery layout (`render_gallery_layout()`) emits a `.avg-gallery-thumb` strip plus one `.avg-tile.avg-gallery-featured`. Thumbs carry `data-label`; they have no `.avg-title` child. Clicking a **thumb** must keep calling `setFeatured` (bound in `initGalleryLayout`); only the **featured** tile opens the lightbox.

```js
  // ============================================================================
  // Lightbox Sequence Collection
  // ============================================================================

  // Filters hide via .is-hidden, pagination via .avg-hidden — two different
  // classes that both resolve to display:none. offsetParent catches both
  // without hardcoding either, and correctly does NOT treat carousel tiles as
  // hidden, since those keep layout boxes and merely scroll out of view.
  function isTileVisible(el) {
    return !!(el && el.offsetParent !== null);
  }

  function readTile(tile) {
    var type = tile.getAttribute('data-type') || 'video';
    var item = {
      type: type,
      tile: tile,
      caption: tile.getAttribute('data-caption') || ''
    };

    if (type === 'image') {
      item.fullUrl = tile.getAttribute('data-full-url') || '';
      var img = tile.querySelector('.avg-thumb-img');
      item.alt = img ? (img.getAttribute('alt') || '') : '';
    } else if (type === 'html') {
      var content = tile.querySelector('.avg-html-content');
      item.html = content ? content.innerHTML : '';
    } else {
      item.provider = tile.getAttribute('data-provider') || '';
      item.videoId = tile.getAttribute('data-video-id') || '';
      item.url = tile.getAttribute('data-url') || '';
    }

    return item;
  }

  function collectSequence(gallery, clicked) {
    // render_gallery_layout() emits a thumb strip plus one featured tile. The
    // strip is the real sequence; a click on the featured tile maps to the
    // currently-active thumb.
    var isGalleryLayout = !!gallery.querySelector('.avg-gallery-thumb');
    var nodes, target = clicked;

    if (isGalleryLayout) {
      nodes = gallery.querySelectorAll('.avg-gallery-thumb');
      if (clicked && clicked.classList.contains('avg-gallery-featured')) {
        target = gallery.querySelector('.avg-gallery-thumb.active') || nodes[0];
      }
    } else {
      // .avg-tile-linked is an <a> and only exists when popup_style is 'none',
      // which never coexists with a lightbox — excluded defensively.
      nodes = gallery.querySelectorAll('.avg-tile:not(.avg-tile-linked)');
    }

    var items = [];
    var startIndex = 0;
    for (var i = 0; i < nodes.length; i++) {
      if (!isTileVisible(nodes[i])) continue;
      if (nodes[i] === target) startIndex = items.length;
      items.push(readTile(nodes[i]));
    }
    return { items: items, startIndex: startIndex };
  }
```

- [ ] **Step 3: Implement the trap**

In the global keydown handler (as rewritten in Task 5), insert this **before** the `if (e.key === 'Escape')` block:

```js
    if (e.key === 'Tab' && lightboxModal && !lightboxModal.hidden) {
      var focusables = [];
      var candidates = lightboxModal.querySelectorAll('button, [href], iframe, input, select, textarea, [tabindex]:not([tabindex="-1"])');
      for (var i = 0; i < candidates.length; i++) {
        if (!candidates[i].hidden && candidates[i].offsetParent !== null) {
          focusables.push(candidates[i]);
        }
      }
      if (!focusables.length) return;

      var first = focusables[0];
      var last = focusables[focusables.length - 1];

      if (e.shiftKey && document.activeElement === first) {
        e.preventDefault();
        last.focus();
      } else if (!e.shiftKey && document.activeElement === last) {
        e.preventDefault();
        first.focus();
      }
      return;
    }
```
