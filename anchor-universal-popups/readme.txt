Create reusable popups with HTML content or YouTube and Vimeo embeds. Trigger by class, ID, or page load with optional delay. Control frequency per session or with a cooldown. Exclude by categories or URLs.

== Description ==
- Custom post type for popups
- Admin code editors with live preview
- YouTube and Vimeo support
- Trigger on page load, on click of class or ID
- Optional delay on page load
- Frequency control: once per session, or cooldown in minutes (uses storage)
- Exclude by URL list or categories

== Usage ==
1. Install and activate
2. Add new Popup in Universal Popups
3. Choose mode (HTML, YouTube, Vimeo)
4. Set trigger and frequency
5. Optionally add exclusions
6. Publish
